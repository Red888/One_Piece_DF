Paramecia type Devil Fruit lists

Gura Gura no Mi

The Gura Gura no Mi is a Paramecia-type Devil Fruit which allows the user to create vibrations, or "quakes", making the user a Tremor Human. It was eaten by Edward Newgate, but upon his death, its power was stolen by Marshall D. Teach.

Current user

Edward Newgate

> More images
Soru Soru no Mi

The Soru Soru no Mi is a Paramecia-type Devil Fruit that allows the user to interact with and manipulate souls. It was formerly eaten by Carmel,[2] but the ability was transferred to Charlotte Linlin after Carmel's death.

Current user

Monkey D. Luffy
